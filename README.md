# DuckLogger

![DuckLogger](https://github.com/user-attachments/assets/50f0e74d-4941-45f8-8832-e5a6bad7b947)

Features:
  1. Records Keystrokes in a log file stored in the internal flash storage
  2. Creates a wifi access point
  3. Log file can be downloaded over the wifi ap from this url: `http://192.168.4.1/`

Componenets used:
  1. ESP32S3 SuperMini
  2. CH9350 Hid Module

Status:
  - Active Development
